SuperSolVR's

This is an interactive virtual reality game that I collaborated on in a four-person group as our final project for Dev Bootcamp. Using the A-Frame library, we created a 3D virtual reality environment on HTML5 that can be rendered from any phone or computer through your browser. However, to get the full experience you would need a VR headset like Google Cardboard. Object behavior was handled using Javascript. Finally, we put the whole project on Rails in order to organize our assets and speed up our app.

The game is designed to be an educational experience tailored for third grade level. Each level focuses on different subjects such as logic, math, and science. Luckily, one of our team members was originally a school teacher and guided us through what material is appropriate for third graders. I was really excited to be a part of this project because of my own passion for video games. In the end, I had a lot of fun building my own environments and watching other people experience it. While we all contributed to each level, I was primarily responsible for level 4 - please check it out below!

MiMoDa

This app uses the OMDB API to provide users with movie information. It built using React and jQuery. It features a simple search bar that returns a list of movies matching the search query. You can then select a movie from the list to see more information about it. This was originally an assigned project at Dev Bootcamp but I went back to this assignment after graduating to add some new functionality and to redesign the UI - making it responsive between different screen sizes and achieving single page functionality.

This was one of my earlier ventures into learning React so working on this project was a roller coaster ride of fun and frustration. After having used Rails, React felt like a completely different animal. Having to pass methods and properties up and down the parent chain felt completely bizarre. It took me a long time to even see what the advantage of using React was. But in the end, it was a great learning experience and stepping stone towards future projects. I'm looking forward to learning React Native for mobile apps and Redux for handling state on larger apps.

Unfortunately, as of May 2017, the OMDB API has gone private so the live website can longer pull data from the API. However, you can still see the codebase below. The screenshot gives a glimpse of what the UI looks like.

Portfolio Helper

This app uses the yahoo-finance and stock-quote gems in order to get real-time stock information and build a fake portfolio. You can create fake portfolios, search for stocks, and add them to your portfolios. The app will use a PostgreSQL database to store portfolio and stock information and update you on the net change of your stock positions since you bought them. You can add more shares of the same stock quote to your portfolio or remove them and the values will update accordingly. It also features secure user authentication using Bcrypt.

The UI is a bit basic but since the last time I worked on this project I have learned to use front-end frameworks like Bootstrap, so I might revisit and redesign the UI sometime in the near future. There is no live website but you can see the codebase below.
